# pydatk-data

[pydatk-data](https://github.com/pydatk/pydatk-data) contains publicly available datasets focused on finance and economics.

The datasets in this repository are from various providers.

Data is redistributed with the permission of the providers.

__You are responsible for checking the data for accuracy and completeness, and checking the provider's terms of usage before using, modifying or distributing this data.__

## Datasets

### The Reserve Bank of New Zealand – Te Pūtea Matua

[The Reserve Bank of New Zealand – Te Pūtea Matua](https://www.rbnz.govt.nz/) is Aotearoa New Zealand's central bank.

Data is retrieved manually from the [Statistical series data files page](https://www.rbnz.govt.nz/statistics/series/data-file-index-page) and redistributed according to the site's [Terms of use](https://www.rbnz.govt.nz/about-our-site/terms-of-use).

The Reserve Bank is acknowledged as the source and copyright owner of the material.

Data is retrieved at the start of each month.

Data is not modified.
