# pydatk-data

[pydatk-data](https://github.com/pydatk/pydatk-data) contains publicly available datasets focused on finance and economics.

See `index.csv` for details.

The datasets in this repository are from various sources. Data is redistributed with the permission of these providers. Attribution and copyright information is included in the index.

__You are responsible for checking the data for accuracy and completeness, and checking the provider's terms of usage before using, modifying or distributing this data.__
